An API made for registering STUDENT type users for the feira-cultural project, that I developed during my senior year of my technical
web development course back in 2025.
It utilizes Node.js frameworks such as cors, dotenv, express and mongoose in order to create student accounts with information about their
name, course, year, etc., and storing all of this information in a MongoDB database. You are able to log in into accounts registered here
by authenticating with the feira-cultural-auth API, as it will pull all information direclty from the MongoDB database.
